# HO3 Sample 3 — Recruiter Prompt Library

## What you'll build
A ready-to-use library of 8 hiring prompts for a recruiter whose job ads and screening notes drift in quality. Each prompt is written in the **CRISP** format (Context, Role, Instruction, Spec, Performance) so Claude gives you consistent, structured drafts — job descriptions, scorecards, interview debriefs and more. Open the page, copy a prompt, paste it into Claude.ai, and you're done.

## Use it with your Claude.ai subscription
No API key needed. Just your normal Claude.ai login.

1. Open the file **`index.html`** from this folder in your web browser (double-click it).
2. You'll see 8 prompt cards (job ads, screening, scorecards, offers and more). Pick the one you need.
3. Click the green **Copy prompt** button on that card.
4. Open **Claude.ai** in another tab and paste the prompt into the chat box.
5. Replace the `[PLACEHOLDERS]` (like `[ROLE TITLE]` or `[CANDIDATE NAME]`) with your real details, then press Enter.
6. Read Claude's draft, tweak it, and use it. If it feels off, tell Claude "make it more inclusive" or "shorter".

No API key needed.

## The prompts in this library
Every card is already filled in and ready to copy:

- **Job Description Writer** — a clear, inclusive job ad from a few notes
- **Screening Question Set** — phone-screen questions tied to the role
- **Candidate Scorecard** — a consistent rubric so interviewers score the same way
- **Interview Debrief** — turn messy notes into a structured assessment
- **Rejection Email** — a kind, respectful "no" that protects your brand
- **Offer Letter Draft** — a warm, clear offer summary
- **Sourcing Boolean String** — a real Boolean search string for LinkedIn/Google
- **Hiring Report Summary** — a pipeline update for the hiring manager

## CRISP format reminder
Every prompt follows this five-part shape, which is why the outputs are so much better:

- **C — Context:** the background situation
- **R — Role:** who Claude should act as
- **I — Instruction:** the core task
- **S — Spec:** format, length, tone constraints
- **P — Performance:** what a great output looks like

## Make it your own
- Swap the `[PLACEHOLDERS]` for one real (anonymised) role you're hiring for.
- Add your company's values to the Spec so the tone matches your employer brand.
- Copy a card's HTML block to add a prompt for your own step, like a take-home brief.
