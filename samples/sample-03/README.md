# HO3 Sample 3 — Recruiter Prompt Library

## Your task

You are a recruiter and your job ads and screening notes are inconsistent. Build 8–12 CRISP prompts for job ads, screening, scorecards and offer letters.

## What to do

1. Look at the prompt areas below — these are the categories your library must cover
2. For each area, write a CRISP prompt (Context, Role, Instruction, Spec, Performance criteria)
3. Test each prompt in Claude.ai and refine it
4. Add your prompts to index.html — a template card is already there as an example
5. Use the copy buttons to use prompts in your workflow

## Prompt areas to cover (8–12 total)

- Job description writer
- Screening question set
- Candidate scorecard
- Interview debrief
- Rejection email
- Offer letter draft
- Sourcing Boolean string
- Hiring report summary

## CRISP format reminder

- **Context:** background situation
- **Role:** who Claude should act as
- **Instruction:** the core task
- **Spec:** format, length, tone constraints
- **Performance:** what a great output looks like
