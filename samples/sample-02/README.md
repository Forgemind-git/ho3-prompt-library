# HO3 Sample 2 — Content Marketer Prompt Library

## What you'll build
A ready-to-use library of 8 content prompts for a solo marketer who reinvents the wheel for every post. Each prompt is written in the **CRISP** format (Context, Role, Instruction, Spec, Performance) so Claude gives you a strong first draft — blog outlines, SEO meta, social repurposing and more — instead of bland filler. Open the page, copy a prompt, paste it into Claude.ai, and you're done.

## Use it with your Claude.ai subscription
No API key needed. Just your normal Claude.ai login.

1. Open the file **`index.html`** from this folder in your web browser (double-click it).
2. You'll see 8 prompt cards (ideation, SEO, newsletter, repurposing, and more). Pick the one you need.
3. Click the green **Copy prompt** button on that card.
4. Open **Claude.ai** in another tab and paste the prompt into the chat box.
5. Replace the `[PLACEHOLDERS]` (like `[TOPIC]` or `[YOUR NICHE]`) with your real details, then press Enter.
6. Read Claude's draft, tweak it, and publish. If the voice is off, just tell Claude "more casual" or "tighten it up".

No API key needed.

## The prompts in this library
Every card is already filled in and ready to copy:

- **Blog Post Ideation** — turn one topic into a list of angles worth writing
- **SEO Title + Meta** — click-worthy titles and meta descriptions for a keyword
- **LinkedIn Carousel Script** — a slide-by-slide script that stops the scroll
- **Email Newsletter** — a warm, skimmable newsletter draft
- **Twitter/X Thread** — a hook-led thread that earns the next line
- **Video Script Outline** — a tight outline for a short video
- **Content Repurposer** — turn one blog post into a week of social posts
- **Performance Debrief** — read your metrics and decide what to do next

## CRISP format reminder
Every prompt follows this five-part shape, which is why the outputs are so much better:

- **C — Context:** the background situation
- **R — Role:** who Claude should act as
- **I — Instruction:** the core task
- **S — Spec:** format, length, tone constraints
- **P — Performance:** what a great output looks like

## Make it your own
- Swap the `[PLACEHOLDERS]` for your real niche, product, and target keyword.
- Add a line to the Spec describing your brand voice (e.g. "playful, no corporate jargon").
- Copy a card's HTML block to add your own prompt — like a case-study or webinar invite.
