What you build

This is your Week-2 CRISP prompt library. You write 8–12 reusable prompts for one real role, each built the CRISP way — Context, Role, Instructions, Specifics, Polish — and each shipped with a before/after example that proves the structured version writes a sharper output. The deliverable is a README index plus the prompt files organised by use-case, so you (or a teammate) can grab the right prompt and get a great result every time. It's peer-reviewed, so keep it clean and well-labelled.
