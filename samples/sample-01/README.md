# HO3 Sample 1 — SDR / Sales Prompt Library

## What you'll build
A ready-to-use library of 8 sales prompts for an SDR who rewrites outreach from scratch every day. Each prompt is written in the **CRISP** format (Context, Role, Instruction, Spec, Performance) so Claude gives you a sharp, on-message first draft instead of a generic one. Open the page, copy a prompt, paste it into Claude.ai, and you're done.

## Use it with your Claude.ai subscription
No API key needed. Just your normal Claude.ai login.

1. Open the file **`index.html`** from this folder in your web browser (double-click it).
2. You'll see 8 prompt cards (prospecting, cold email, follow-up, and more). Pick the one you need.
3. Click the green **Copy prompt** button on that card.
4. Open **Claude.ai** in another tab and paste the prompt into the chat box.
5. Replace the `[PLACEHOLDERS]` (like `[PROSPECT NAME]` or `[YOUR PRODUCT]`) with your real details, then press Enter.
6. Read Claude's draft, tweak it, and use it. If the tone is off, just tell Claude "make it warmer" or "make it shorter".

No API key needed.

## The prompts in this library
Every card is already filled in and ready to copy:

- **Prospect Research Brief** — turn a LinkedIn/website into pain points and a hook
- **Cold Email First Line** — a personalised opener that earns the next sentence
- **Follow-up Sequence** — a polite multi-touch chase that isn't annoying
- **Objection Reframe** — handle "too expensive / no time / using a competitor"
- **LinkedIn Connection Note** — a short, human connection request
- **Call Prep Sheet** — a one-page brief before a discovery call
- **Win/Loss Debrief** — capture why a deal closed or slipped
- **Pipeline Update Summary** — a clean status update for your manager

## CRISP format reminder
Every prompt follows this five-part shape, which is why the outputs are so much better:

- **C — Context:** the background situation
- **R — Role:** who Claude should act as
- **I — Instruction:** the core task
- **S — Spec:** format, length, tone constraints
- **P — Performance:** what a great output looks like

## Make it your own
- Swap the `[PLACEHOLDERS]` for a real (anonymised) prospect from your CRM.
- Add a line to the Spec like "match our brand voice: confident but never pushy".
- Copy a card's HTML block to add your own 9th or 10th prompt to the library.
