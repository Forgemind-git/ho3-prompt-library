# HO3 Sample 5 — People Manager Prompt Library

## What you'll build
A ready-to-use library of 8 management prompts for a new people-manager who stares at a blank page before every 1:1 and review. Each prompt is written in the **CRISP** format (Context, Role, Instruction, Spec, Performance) so Claude gives you fair, specific, humane drafts — 1:1 agendas, reviews, feedback, coaching and more. Open the page, copy a prompt, paste it into Claude.ai, and you're done.

## Use it with your Claude.ai subscription
No API key needed. Just your normal Claude.ai login.

1. Open the file **`index.html`** from this folder in your web browser (double-click it).
2. You'll see 8 prompt cards (1:1s, reviews, feedback, coaching and more). Pick the one you need.
3. Click the green **Copy prompt** button on that card.
4. Open **Claude.ai** in another tab and paste the prompt into the chat box.
5. Replace the `[PLACEHOLDERS]` (like `[REPORT NAME]` or `[PASTE NOTES]`) with your real details, then press Enter.
6. Read Claude's draft, tweak it, and make it yours. If it feels off, tell Claude "warmer" or "more direct".

No API key needed.

## The prompts in this library
Every card is already filled in and ready to copy:

- **1:1 Agenda Generator** — a focused agenda for your next one-on-one
- **Performance Review Draft** — a balanced review from your notes and examples
- **Constructive Feedback Framer** — fair, specific feedback (behaviour, not character)
- **Goal-Setting Session** — clear, motivating goals you set together
- **Coaching Question Set** — open questions that help your report think it through
- **Promotion Case Writer** — a structured case for a promotion
- **PIP Summary** — a fair, humane improvement-plan outline (have HR review it)
- **Team Health Check** — read the room and spot what needs attention

## CRISP format reminder
Every prompt follows this five-part shape, which is why the outputs are so much better:

- **C — Context:** the background situation
- **R — Role:** who Claude should act as
- **I — Instruction:** the core task
- **S — Spec:** format, length, tone constraints
- **P — Performance:** what a great output looks like

## Make it your own
- Swap the `[PLACEHOLDERS]` for one real (anonymised) report and situation.
- Add your team's values to the Spec so the tone matches how you lead.
- Copy a card's HTML block to add a prompt for your own ritual, like a skip-level prep.

> Note: drafts about people are a starting point, not a verdict. Always review, edit, and — for anything sensitive like a PIP — loop in HR before sharing.
