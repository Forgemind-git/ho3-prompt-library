# Prompt 10 — YouTube Description

**Library:** Content Marketer Prompt Library  
**Role:** Content Marketing Manager

---

## Purpose

Write a YouTube video description optimised for search discovery (YouTube SEO), with timestamps for navigation, relevant keywords placed naturally, and a structured CTA section — because most YouTube descriptions are an afterthought that costs the video half its potential reach.

---

## CRISP Breakdown

| Element | Content |
|---------|---------|
| **Context** | Video: [title and topic]. Channel: [channel name and niche]. Target keyword: [what people would search to find this]. Video length and timestamps: [list the sections and their start times]. Links to include: [resources, products, other videos]. |
| **Role** | You are a YouTube SEO specialist who knows that a description is not a transcript summary — it's a search document and a navigation tool for the viewer. |
| **Instruction** | Write a YouTube description with: keyword-rich opening paragraph (first 2–3 lines visible before "show more"), a what-you'll-learn section, timestamped chapters, resources section, and a CTA block. |
| **Spec** | YouTube description format. Opening paragraph 150 words (appears before "show more"). Timestamps in the standard YouTube format (0:00, 1:30, etc.). Total length: 300–400 words. Primary keyword in first 25 words. |
| **Performance** | The video gets discovered by search for the target keyword. Viewers navigate using the timestamps. One resource link or CTA is clicked by at least 3% of viewers. |

---

## Full Prompt

```
Context: I'm writing a YouTube description for a video on [Company/Channel Name]'s channel. 
- Video title: [exact title as it appears on YouTube]
- Topic summary: [2–3 sentences about what the video covers]
- Target keyword: [what people would search to find this video]
- Video length: [total minutes]
- Timestamps and sections: [list each section with its start time — e.g., "0:00 Intro / 2:15 The CSAT problem / 8:30 The alternative metrics / ..."]
- Resources to link: [blog post, template, product page, related video — give titles and URLs]
- Subscribe/social CTA: [your channel subscribe prompt and any social handles]
- Primary CTA goal: [what you most want viewers to do after watching]

Role: Act as a YouTube SEO specialist who treats video descriptions as discovery documents, not summaries. The first 150 words are indexed by YouTube's algorithm and set viewer expectations — they're too valuable to waste.

Instruction: Write a YouTube description with:
1. Opening Paragraph (150 words, before "show more") — keyword in first 25 words, explain the core value prop, tell the viewer what they'll learn, written to be read (not a keyword dump)
2. What You'll Learn — 4–6 bullet points of key takeaways
3. Timestamps — all chapters in standard YouTube format
4. Resources — labelled links with a one-line description of each
5. CTA Block — subscribe ask, social follow, primary action link

Spec:
- YouTube description format
- Opening paragraph: 150 words max (before "show more" fold)
- Timestamps: standard format (0:00, 1:30, 10:45 etc.)
- Primary keyword in first 25 words
- Total length: 300–400 words
- CTA block at the end (most viewers don't see it; that's okay — it's for the ones who scroll)

Performance bar: Video ranks on page 1 for the target keyword within 30 days of publish. Timestamps are accurate and cover every major section. One external link gets clicked.
```

---

## Before Example (Weak Prompt)

**Prompt used:**
> Write a YouTube description for a video about customer support metrics.

**Output produced:**
> In this video, we talk about customer support metrics and how to improve them. Watch to learn more about how to measure customer satisfaction. Like and subscribe for more content!

**Why it's weak:** No keyword placement strategy, no timestamps, no value-specific bullet points, "Watch to learn more" is meaningless, "Like and subscribe" is the entire CTA block. This description does nothing for search discovery.
